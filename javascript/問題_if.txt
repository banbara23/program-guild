■ ifのカッコ内について

この前はifを使ってプログラムを書くことを覚えたよね

if (x=='abc') {
 // xがabcなら実行
}
else {
 // それ以外はここを実行
}

このifのかっこの中で何が行われているか？
それを確かめるには、以下を実行してみるといい

var x = 'abc'; // 変数xには文字列'abc'が格納されている状態
console.log(x=='abc'); // ifの中身をログに出力すると

true

と出力されるはず。
このtrueとは何かというと、boolean(ブーリアン)という特殊な「型」と呼ばれるもの。
このboolean型はとても大事だから、確実に理解して覚えてほしい。

■ booleanとは？

booleanには
true (トルゥー)と、
false (ファルス) 
の２パターンしかない。

どういう意味かというと 
true「真」
false「偽」
とよく参考書に説明されているが、分かりやすく言うと
true 「条件を満たしている」
false 「条件が満たされていない」
と考えればよい。

ここでifのカッコの中の話に戻るけど、
if (x==1)
これはxが1ならtrue、1じゃないならfalseとなる。

■ 結論

つまり、ifのカッコの中はboolean型が使われており、そのtrue/falseによって処理が分岐されてくる。

ちなみに、booleanは変数に入れることもできる
var x = true;
console.log(x) //-> true
var y = false;
console.log(y) //-> false

ここまで読んで理解できたかな？

ifについて問題①
この前はifで文字を判定したので、今回は数値を判定しよう。
以下のxの値を自分で色々な数値に変更して実行してみてくれ

var x = 3;
if (x == 3) { // 値を指定するとき
 console.log('xは3です')
}

if (x != 2) { // 〜以外の時と判定
 console.log('xは2以外です')
}

if (x > 3) { // 〜より大きいと判定する時
 console.log('xは3より大きい')
}

if (x < 4) { // 〜より小さいと判定する時
 console.log('xは4より小さい')
}

if (x >= 5) {
 console.log('xは5以上です')
}

if (x <= 6) {
 console.log('xは6以下です')
}
